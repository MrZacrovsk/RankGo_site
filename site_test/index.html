As distros Linux mais simples de usar

Olá pessoal, hoje estou trazendo uma pequea lista das distros que tive mais produtividade e que acredito que vocês também terão. Lembrando que não vai ser um rank de pior ou melhor, apenas as que mais tive produtividade.☺

Linux Mint

Linux Mint foi pra mim a distro onde tive mas produtividade nos ultimos meses, extremamente leve, simples de usar, e muito bem organizado.

Ubuntu Budgie

Um versão do Ubuntu com o Budgie Desktop, é a distro mais parecida com Mac, muito leve e simples de usar, recomendo.

Pop OS

Tá ai uma distro com gnome que eu sou apaixonado, pro meu note é um pouco pesado mais nada que te tire do serio e teho certeza que você irá gostar dela.

Manjaro

Uma distro diferente das já citadas, mais que mantém a simplicidade a maior difereça é ns comandos que você vai ter que tirar um tenpo pra pegar o jeito.
